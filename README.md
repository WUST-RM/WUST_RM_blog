# 武汉科技大学RoBomaster崇实战队算法组WUST_RM博客

## 博客发布说明

-  加入WUST_RM组织
-  加入组织内Team:blog以拿到编写权限
-  clone[本仓库](https://github.com/WUST-RM/WUST_RM_blog)，若clone过,记得在发布新blog前pull来更新别人的blog
-  在src/content/posts目录下创建一个新的文件夹,具体格式可参考其他blog或文档[传送门](https://docs.mizuki.mysqil.com/press/folder/)
-  commit and push
-  记得检查是否发布成功（需要等一会）
